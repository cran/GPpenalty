# GPpenalty 1.0.0
## Major changes
* DPE and kernel functions have been added.
* DPE and MD metric options have been added in gp_cv() function. 

## Minor changes
* User can specify the upper and lower bounds for the lengthscale parameter for the optimization procedure.
* An option has been added to control the number of randomly generated initial value sets to evaluate. The default is 10.
